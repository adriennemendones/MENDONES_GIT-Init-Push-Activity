ALL ABOUT RASPBERRY PI

The Raspberry Pi is a very cheap computer that runs Linux,
but it also provides a set of GPIO (general purpose input/output) pins, allowing you to control electronic components
for physical computing and explore the Internet of Things (IoT).

Raspberry Pi is the name of a series of single-board computers made by the Raspberry Pi Foundation,
a UK charity that aims to educate people in computing and create easier access to computing education.
